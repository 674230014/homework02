Workshop: Various Links

เรียนรู้ลิ้งค์หลายรูปแบบ พร้อมตัวอย่างแบบฝึกหัด

1.External Links

Visit Wikipedia

target="_blank" จะเปิดลิ้งค์ในแท็บใหม่

Visit Google

target="_self" จะเปิดลิ้งค์ในแท็บเดิม

2.Internal Links & Anchor Links

<a href="about.html">About Us</a> 
<a href="#jump">Jump to Section</a> 
<h3 id="jump">This is the jump target</h3>
Click to Jump

This is the jump target

3. Email Links

<a href="mailto:student@example.com?subject=Hello"> 
Send an Email 
</a>
Send an Email

4. Download Links

<a href="files/sample.txt" download>Download File</a>
Download Text File

ต้องมีไฟล์ในโฟลเดอร์จึงจะดาวน์โหลดได้

5. Button Links

ลิ้งก์ที่ออกแบบคล้ายปุ่ม เหมาะกับปุ่มสมัคร ปุ่มอ่านเพิ่มเติม หรือปุ่มนำทาง

<a href="https://www.example.com" class="btn-link"> 
Visit Website 
</a> 

<button onclick="document.location='index.html'">HTML Tutorial</button> - มักใช้เมื่อลิงก์ไป หน้าในเว็บเดียวกัน - ผูกกับ DOM (document) โดยตรง 
<button onclick="location.href='https://www.example.com'">Visit Website</button> - นิยมใช้มากกว่า - เหมาะทั้งลิงก์ ภายในและภายนอกเ
